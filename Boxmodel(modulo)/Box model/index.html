Box Model

Box Model é um modelo de caixa que define como as propriedades de CSS são aplicadas a um elemento HTML. O modelo de caixa é composto por quatro partes: conteúdo, padding, border e margin.

  1. Conteúdo: é a área onde o conteúdo do elemento é exibido.

  2. Padding: é a área entre o conteúdo e a borda do elemento.

  3. Border: é a borda do elemento.

  4. Margin: é a área entre a borda do elemento e os elementos vizinhos.
